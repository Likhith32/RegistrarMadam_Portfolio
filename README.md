JNTU-GV Registrar Portfolio
A modern, responsive portfolio website for the Registrar of JNTU-GV (Jawaharlal Nehru Technological University, Kakinada - GV Campus). This React-based application showcases achievements, awards, contributions, and professional information with admin editing capabilities.
Features

Dynamic Content Management: Admin can edit all content without changing code
Comprehensive Sections:

Professional Information & Biography
Academic Achievements
Awards & Recognition
Contributions to Institution
Publications & Research
Administrative Initiatives


Responsive Design: Works seamlessly on desktop, tablet, and mobile
Admin Panel: Secure login for content updates
Modern UI: Clean, professional interface built with React

Tech Stack

Frontend: React 18+
Styling: Tailwind CSS
State Management: React Context API
Routing: React Router
Icons: Lucide React
Authentication: JWT-based admin authentication
